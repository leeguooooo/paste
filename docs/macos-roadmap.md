# macOS App Roadmap (Paste 对标)

## 目标

- 主平台：macOS
- 跨端策略：桌面端统一走 Electron，Windows 兼容但不优先
- 设计方向：参考 Paste 的信息密度、快速检索、低干扰交互

## 已落地（M0）

- Electron + React 基础工程：`apps/macos`
- API 联调：列表、新增、收藏、删除
- 全局快捷键：`Cmd/Ctrl + Shift + V`（显示/隐藏主窗口）
- 剪贴板监听：后台轮询并自动入库（可开关）
- 托盘菜单：显示/隐藏、手动抓取、退出

## M1（核心可用）

- 历史列表增强：按类型徽标、快速预览、快捷复制
- 键盘优先：方向键导航、回车复制、ESC 收起
- 搜索/收藏/标签筛选完整映射
- 启动项与开机自启

## M2（Paste 体验）

- Quick Paste 面板（悬浮 + 热键唤起）
- 最近使用排序与固定条目（pin）
- Snippets 分组与模板插入
- 多设备同步状态可视化

## M3（进阶）

- 本地索引与离线缓存（SQLite）
- OCR/图片文本提取（可选）
- AI 清洗与摘要（可选）

## 约束说明

- 将参考 Paste 的交互与效率设计，不复制其品牌资源与受保护素材。
